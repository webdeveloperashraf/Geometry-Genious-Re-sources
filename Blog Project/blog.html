Question No-01: What is DOM? What is the purpose of DOM?

Answer:The complete form of DOM is-Document Object Model.it transforms the html document into an object. It connects the html page to the js page and acts as a programming interface.

The purpose of dom is to establish a communication between one programming language and another language and to modify, expand, add, remove the structure of the structure and work as a programming interface.
